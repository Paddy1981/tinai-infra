Quick Start Guide
-----------------

1. Install the TinAI CLI
   npm install -g @tinai/cli

2. Connect your account
   Run: tinai login

3. Deploy your first app
   Navigate to your project and run: tinai deploy

4. Manage from the dashboard
   Monitor deployments, configure domains, and manage your team
